# Data Directory

Place the Kaggle competition data files here:

- `train.csv`
- `test.csv`
- `data_description.txt`
- `sample_submission.csv`

Download from: https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques/data

Or use the Kaggle CLI:
```bash
kaggle competitions download -c house-prices-advanced-regression-techniques -p .
unzip house-prices-advanced-regression-techniques.zip
```
